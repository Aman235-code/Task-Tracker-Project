# Task Tracker Project Using Reactjs

This is a simple Task Tracker web app built with **React.js** and **Tailwind CSS**.  
It lets you add tasks, view all tasks, and mark tasks as completed.

## Features

- Add a new task
- See all tasks in a list
- Mark tasks as completed or pending
- Simple and clean UI with Tailwind

## Technologies Used

- React.js
- Tailwind CSS
- Vite (build tool)

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have **Node.js** and **npm** installed.

### Install Dependencies

Clone the repo:

```bash
git clone https://github.com/Aman235-code/Task-Tracker-Project.git
```

Go into the project folder:

```bash
cd Task-Tracker-Project
```

Install the required packages:

```bash
npm install
```

Start the App

```bash
Run the development server:
```

```bash
npm run dev
```

## How to Use

- Type a task in the input
- Click the Add button
- See the task added to the list
- Click on a task status to toggle between Pending and Completed
